# Lab-Python 🐍

A collection of foundational Python scripts developed during my B.Tech lab coursework. This repository covers core programming concepts, control structures, and modular programming, building a strong base for advanced development and data handling.

## 📂 Repository Structure & List of Experiments

### [Experiment 1: Python Basics & Command Line Execution](./Exp_01_Basics)
*Installation of Python, verifying PATH, and executing scripts.*
* `1_cmd_args_display.py`: Takes input text as a command-line argument and displays it on the screen.

### [Experiment 2: Mathematical Operations](./Exp_02_Math_Operations)
* `2a_cmd_args_sum.py`: Takes 2 numbers as command-line arguments and prints their sum.
* `2b_gcd.py`: Calculates the Greatest Common Divisor (GCD) of 2 numbers.
* `2c_exponentiation.py`: Finds the exponentiation (power) of a number.
* `2d_simple_calculator.py`: A basic calculator for standard arithmetic operations.

### [Experiment 3: Number Theory & Series](./Exp_03_Number_Theory)
* `3a_factorial.py`: Finds the factorial of a given number.
* `3b_fibonacci_series.py`: Evaluates the Fibonacci series up to a given number 'n'.
* `3c_armstrong_number.py`: Checks if a given number is an Armstrong number.
* `3d_sum_of_n.py`: Calculates the sum of first N natural numbers.

### [Experiment 4: Control Flow & Modules](./Exp_04_Control_Flow_Modules)
* `4a_countdown.py`: Takes a number as input and prints a countdown to zero using a `while` loop.
* `4b_circulate_n.py`: Finds and displays circulating 'n' values.
* `4c_import_module.py`: Implements a module using the standard `import` statement.
* `4d_from_import.py`: Implements a module using the `from...import` statement.

### [Experiment 5: Algorithmic Problem Solving](./Exp_05_Algorithms)
* `5a_sum_of_primes.py`: Calculates the sum of all prime numbers below two million.
* `5b_even_fibonacci_sum.py`: Finds the sum of the even-valued terms in the Fibonacci sequence whose values do not exceed four million.
