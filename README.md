# Lab-Python 🐍

A collection of foundational Python scripts developed during my B.Tech lab coursework. This repository covers core programming concepts, control structures, and modular programming.

## 📂 Repository Structure & List of Experiments

### [Experiment 1: Python Basics & Command Line Execution](./Exp1_CMD_Print)
*Installation of Python, verifying PATH, and executing scripts.*
* `1_a.py`: Takes input text as a command-line argument and displays it on the screen.

### [Experiment 2: Mathematical Operations](./Exp2_basic_calculations)
* `2_a.py`: Takes 2 numbers as command-line arguments and prints their sum.
* `2_b.py`: Calculates the Greatest Common Divisor (GCD) of 2 numbers.
* `2_c.py`: Finds the exponentiation (power) of a number.
* `2_d.py`: A basic calculator for standard arithmetic operations.

### [Experiment 3: Number Theory & Series](./Exp3_series_operations)
* `3_a.py`: Finds the factorial of a given number.
* `3_b.py`: Evaluates the Fibonacci series up to a given number 'n'.
* `3_c.py`: Checks if a given number is an Armstrong number.

### [Experiment 4: Control Flow & Modules](./Exp4_count_import)
* `4_a.py`: Takes a number as input and prints a countdown to zero using a `while` loop.
* `4_b.py`: Finds and displays circulating 'n' values.
* `4_c.py`: Implements a module using the standard `import` statement.
* `4_d.py`: Implements a module using the `from...import` statement.
* `mymodule.py`: Custom module imported by `4_c.py` and `4_d.py`.

### [Experiment 5: Algorithmic Problem Solving](./Exp5_Algorithms)
* `5_a.py`: Calculates the sum of all prime numbers below two million.
* `5_b.py`: Finds the sum of the even-valued terms in the Fibonacci sequence whose values do not exceed four million.
